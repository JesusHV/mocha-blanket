# mocha-blanket

Handler to use mocha with blanket js

## Usage
```js
<script src="{pathModule}/index.js"></script>
<script>
  describe('Awesome tests', () =>{
    it('Is a test', (done) => {
      done();
    });
  });
</script>
```

### This script depends of [Blanket JS](http://blanketjs.org/) and [Mocha JS](https://mochajs.org/)
